AUTOSTREAM[ 06 / ARCHITECTURE ]
Engineering decisions

Five nodes. Three locks.
Zero hallucinated prices.

01

LangGraph state machine, not a prompt pile.

classify_intent → extract_lead → retrieve → capture → respond. Routing edges enforce the phase contract in code, not vibes.

02

FAISS RAG fires only on product inquiries.

all-MiniLM-L6-v2 embeddings, pre-built cache committed to the repo. Greetings and slot-collection turns skip retrieval entirely.

03

Lead capture has three locks.

Tool can only fire when: all 3 slots filled, email regex-valid, and user has confirmed. Premature firing is structurally impossible.
