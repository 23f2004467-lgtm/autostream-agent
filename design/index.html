[ 02 / NOTES ]

Design decisions.

[ ROUTING ]

Phase, not intent.

Routing is driven by phase, not a per-turn intent classifier. Stops "Jainam" from being mis-read as a greeting mid-collection.

[ TOOL GATE ]

Three locks on capture.

mock_lead_capture fires only if all 3 slots are filled, email regex-valid, and user has explicitly confirmed. Premature firing is structurally impossible.

[ RAG ]

Retrieval only when asked.

FAISS retrieval fires on product_inquiry only — greetings, slot-collection, and objections skip it. Cheaper and less prone to answer-drift.
